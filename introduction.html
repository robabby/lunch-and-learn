Lesson 1-1: Selectors

CSS selectors can range from the incredibly simple to the overly complex.


First, let's take a look at some plain HTML

Now lets style our example HTML with some basic CSS selectors: {% highlight css %} div { padding:5px; border:5px solid #09f; background-color:#eee; } h1 { margin-bottom:15px; font-family:Arial; font-size:24px; text-decoration:underline; font-variant:small-caps; } p { font-family:Arial; font-size:14px; margin-bottom:10px; } a { font-family:Arial; color:#09f; text-decoration: none; } {% endhighlight %}

View the result 


Next we have some more advanced selectors: {% highlight css %} a.outbound[href^="http"] { background-image:url(../img/some_image.png); baclkground-position:0px 0px; background-repeat:no-repeat; } ul.nav > li a[data-*="foo"] { font-weight:bold; } {% endhighlight %}
